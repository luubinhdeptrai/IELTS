# Special Note: `no` vs `not`

Bạn hỏi: **khi nào dùng `no`, khi nào dùng `not`?**

## 1. `no + noun`

Dùng **no** trực tiếp trước danh từ.

Cấu trúc:

- **no + noun**

Examples:

- There is no error.
- There is no appropriate plan.
- We have no evidence.
- The system has no backup mechanism.

Nghĩa:

- không có + danh từ

---

## 2. `not + adjective / verb / adverb / article + noun`

Dùng **not** để phủ định động từ, tính từ, trạng từ, hoặc cụm có `a/an/the`.

Examples:

- The system is not stable.
- This solution is not appropriate.
- He does not understand the problem.
- There is not an appropriate plan.
